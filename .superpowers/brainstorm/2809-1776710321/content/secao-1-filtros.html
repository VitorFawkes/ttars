"Meu painel" leva ao dashboard do papel do usuário logado (role do profile). Se for admin/dono, vai pro Dono; se for Planner, vai pro Vendas; se for Concierge, vai pro Pós; etc.

Abaixo, acesso livre aos 5 dashboards (admin/dono vê todos; operador vê o dele + os dashboards macro do time).

"Explorar" é a ferramenta livre de slice-and-dice — cruzar qualquer métrica com qualquer filtro sem curadoria.

Permissão: cada persona só vê dashboards compatíveis com seu papel. Dono/admin vê tudo.

"+ Mais filtros" expande:

Origem ▾ (mkt, indicação, carteira, site...)
Entry path ▾ (SDR completo / Direto Planner / Retorno / Indicação)
Destino ▾
Tags ▾
Modo (Entradas/Coorte/Ganho) ▾

Sempre visíveis (3): Período, Fase, Dono — os essenciais.

No "+ Mais filtros" (5): Origem, Entry path, Destino, Tags, Modo. Aparecem expandidos quando aplicados.

Persistência na URL: toda combinação de filtros vira querystring. Dá pra mandar link no WhatsApp: "olha essa tela". Dá pra salvar favorito.

Defaults são ponto de partida — user pode sobrescrever e salvar uma "view favorita" (fase 2).

Como o cross-cut "Pós com origem Travel Planner" se realiza

Passo 1: User abre 🛠️ Pós-Venda (defaults: Fase=Pós, Período=Próx. 30d)

Passo 2: Clica em "+ Mais filtros" → Entry path → seleciona "Direto Planner"

Passo 3: Todos os gráficos e KPIs do dashboard Pós se reajustam. A URL passa a ser /analytics/pos?period=30d&entry=direct_planner — compartilhável.

Passo 4: Se quiser comparar, troca Entry path pra "SDR completo" — mesma tela, outra leitura.

A chave: o dashboard é curado (Pós tem os widgets certos pra Pós), mas os filtros cruzam qualquer dimensão. Você nunca fica preso a uma visão.
