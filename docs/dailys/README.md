# Dailys — ttars

Anotações e backlog das dailys do time do ttars. Cada arquivo aqui é o
registro bruto de uma reunião diária: decisões, próximos passos e itens
que vão virar tarefa no CRM ou ficar pendentes para um lote posterior.

## Convenções

- **Nome do arquivo:** `daily-DDMM.md` (ex: `daily-0705.md` = daily de 7 de maio).
- **Conteúdo:** texto livre, organizado por tópicos. Sem template rígido —
  o objetivo é capturar contexto, não preencher formulário.
- **Quem escreve:** quem conduziu a daily ou tirou nota. Os itens são
  atribuídos ao dono pelo nome no título da seção quando relevante.
- **Status:** o arquivo é o snapshot do dia. Atualizações de execução
  acontecem no CRM (cards, propostas, PRs), não aqui.
