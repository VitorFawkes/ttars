E o Obsidian?

Pesquisado a fundo (2 vezes): não vale como cérebro dos agentes. O agente lê arquivos organizados, não "grafo" — e o projeto já faz o padrão que a Anthropic recomenda. Obsidian ajuda humano, não o robô.

Quer mesmo assim?

Como a pasta de memória já é compatível, você pode abri-la no Obsidian só para você navegar visualmente. Zero trabalho de engenharia, zero impacto nos agentes. Só se quiser.

Base: 2 pesquisas profundas (43 fontes, ~200 afirmações verificadas com checagem adversarial) + mapa do projeto conferido diretamente em Opus 4.8.
